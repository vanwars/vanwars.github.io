I am an Assistant Professor of Instruction in the Department of Computer Science at Northwestern University. I teach courses in human-computer interaction (HCI), ethics, introductory programming, and web development. I also co-direct the TREE Lab with Sepehr Vakil.

I am an educator...My current research examines two broad themes: (1) the possibilities and limits of data-centric approaches to grassroots civic advocacy; and (2) how to design inclusive learning environments that foster robust computing and data literacies.

I later received my PhD in Information Science at the UC Berkeley School of Information, where I studied some approaches for engaging young people in grassroots environmental justice efforts, including some of the ways in which data and multimedia in advocacy and citizen science.

Prior to academia, I worked as a software designer/developer, building data and communication systems to support public accountability for city and regional development projects.

CS 110: Intro to Computer Programming with Python

The goal of CS110 is twofold: (1) to learn some fundamental programming constructs and techniques for marshaling them, and (2) to practice applying these ideas range of problem-solving and creative pursuits (e.g. animations, music, data-analysis, building data-driven apps). The course assumes no prior programming knowledge.

MMART 162: Intro to Computer Programming using JavaScript

The goal of MMART 162 was to give students an introduction to programming within a web context. The course began with an introduction to JavaScript programming fundamentals – variables, lists, loops, events, and functions. Students then used these concepts to create a variety of mini-projects, including web animations, interactive user interfaces, and ultimately a stand-alone web application and web server. Intro to Web Design

This course was designed for a broad audience -- from high school students to seniors. In it, we explored some HTML & CSS fundamentals, examined how servers, browsers, search engines, IP addresses, and DNS worked, and practiced instantiating various web design principles. As a final project, each student created their own website and designed a custom WordPress theme.
